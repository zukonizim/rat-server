## Desteklenen Sürümler

Bu bölümü, projenizin hangi sürümlerinin şu anda güvenlik güncellemeleriyle desteklendiğini insanlara bildirmek için kullanın.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Bir Güvenlik Açığını Bildirme

Bu bölümü, kişilere bir güvenlik açığını nasıl bildireceklerini anlatmak için kullanın.

Nereye gitmeleri gerektiğini, bildirilen bir güvenlik açığı hakkında ne sıklıkla güncelleme alabileceklerini, güvenlik açığı kabul edilirse veya reddedilirse ne beklemeleri gerektiğini vb. anlatın.
